## 📱 Tentang Aplikasi
**LifeBar** adalah sebuah aplikasi *habit tracker* berbasis *gamifikasi Role-Playing Game* (RPG). Aplikasi ini dirancang untuk memotivasi pengguna menyeimbangkan produktivitas kerja dan kesehatan fisik melalui sistem manajemen energi virtual.

## ✨ Fitur Utama
1. **Sistem Quest (Misi):** Terbagi menjadi *Main Quest* (tugas berat), *Side Quest* (tugas ringan), dan *Daily Quest* (istirahat/pemulihan).
2. **Manajemen Energi (MP):** Menyelesaikan tugas berat akan menguras MP. Jika MP habis (0), sistem memblokir misi berat dan mewajibkan pengguna melakukan pemulihan (*Daily Quest*).
3. **Sistem Level & Ekonomi:** Pengguna mendapatkan EXP untuk naik level otomatis dan Koin virtual setiap kali menyelesaikan misi.
4. **Toko (Shop):** Koin yang dikumpulkan dapat ditukar dengan *reward* di dunia nyata.
5. **Animasi Real-Time:** Menggunakan `ObjectAnimator` pada Android Studio untuk pergerakan *bar* HP, MP, dan EXP yang *smooth* dan dinamis.
6. **Desain UI 5 Halaman:** Meliputi halaman *Splash*, Layar Utama, Daftar Misi, Toko, dan Profil Pemain.

## 🛠️ Teknologi yang Digunakan
* **Bahasa Pemrograman:** Kotlin & XML
* **IDE:** Android Studio
* **Penyimpanan:** SharedPreferences (Penyimpanan Lokal)
